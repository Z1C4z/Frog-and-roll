Sobre o Projeto

Frog And Roll é um jogo de plataforma 2D com ritmo musical de código aberto criado como um projeto de conclusão de curso no Senac Minas - Divinópolis.

Controle nosso sapo protagonista através de desafios musicais, coletando itens e desviando de obstáculos ao ritmo da música. Cada nível apresenta uma trilha sonora única que dita o ritmo dos desafios.
